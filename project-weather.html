Weather App

An interactive weather dashboard that fetches real-time weather data from the OpenWeatherMap API. It displays current conditions, humidity, wind speed, and a 5-day forecast.


Technologies Used

  • JavaScript (Async/Await)
  • OpenWeatherMap API
